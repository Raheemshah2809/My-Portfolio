Breakout Game
This is the game I made, it was a take on Atari Breakout.

About the game.

This is my first Project I have worked on and my first project with Java, This Game is a take on the classic Atari breakout and I showed of my skills to make a version my self, I started off with a blank canvas and added the bricks, paddle, ball ,score and music.

How does the game work.

The game works by the user using the arrow keys to move the paddle and using the ball to hit the bricks with the ball getting faster as the score increases, you also have 3 lives to complete the game.
